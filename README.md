# 👋🏻 프로젝트 소개

1. 시각장애인의 옷의 색상, 종류, 패턴 인식의 어려움
2. 시각장애인의 옷의 위치 파악의 어려움
3. 시각장애인의 옷 정보 체계적 관리 어려움

위와 같은 어려움을 겪는 시각 장애인을 위해 온디바이스 AI와 IoT 기술을 결합한 **스마트 AI 옷장 서비스**를 기획하게 되었습니다.

## 🌟 주요 기능

- 옷 위치, 옷 정보 자동 저장
- 원하는 옷 위치 안내
- 나만의 코디 저장
- NFC 태깅을 통한 편리한 옷 정보 파악
